## 856. 括号的分数
给定一个平衡括号字符串 S，按下述规则计算该字符串的分数：

() 得 1 分。
AB 得 A + B 分，其中 A 和 B 是平衡括号字符串。
(A) 得 2 * A 分，其中 A 是平衡括号字符串。
 

示例 1：
```
输入： "()"
输出： 1
````
示例 2：
```
输入： "(())"
输出： 2
```
示例 3：
```
输入： "()()"
输出： 2
```
示例 4：
```
输入： "(()(()))"
输出： 6
```
***
#### 思路：
只有()会贡献分数，所以只需要计算每个()的深度即可。
```
class Solution:
    def scoreOfParentheses(self, s: str) -> int:
        ans = d = 0
        for i, c in enumerate(s):
            if c == '(':
                d += 1
            else:
                d -= 1
                if s[i - 1] == '(':
                    ans += 1 << d
        return ans
```
